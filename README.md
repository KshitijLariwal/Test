# Test Repository

This repository contains a collection of web development assignments demonstrating fundamental concepts in HTML, CSS, and JavaScript.

## Repository Structure

- **Assignment - 3/**: Check Number Assignment
  - A simple web application that prompts the user for a number and determines if it's positive, negative, or zero using JavaScript.
  - Technologies: HTML5, JavaScript
  - Files: `Assignment-3.html`, `README.md`

- **Assignment - 4/**: DOM Manipulation Tasks
  - **Task-1**: Dynamic button creation and insertion using JavaScript DOM methods.
  - **Task-2**: Element selection, direct styling, and CSS class manipulation.
  - Technologies: HTML5, CSS3, JavaScript
  - Files: `Task-1/` (index.html, script.js), `Task-2/` (index.html, index.js, style.css), `README.md`

## Purpose

These assignments are designed to build foundational skills in:
- HTML structure and semantics
- JavaScript DOM manipulation
- CSS styling and transitions
- Event handling and user interaction

Each assignment includes detailed README files with usage instructions and key concepts covered.
